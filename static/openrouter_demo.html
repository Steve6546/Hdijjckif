شرح استخدام OpenRouter مع نظام الوكلاء الذكي

دليل المطور لاستخدام النماذج المجانية والمدفوعة

مقدمة

يدعم نظام الوكلاء الذكي الآن استخدام OpenRouter للوصول إلى مجموعة متنوعة من نماذج الذكاء الاصطناعي، بما في ذلك:

  • النماذج المجانية: مثل Google Gemini و Llama 3.3 و Qwen 2.5
  • النماذج المدفوعة: مثل GPT-4 و Claude 3 و Mistral Large (تتطلب مفتاح API مدفوع)

كيفية استخدام النماذج المجانية

يمكنك استخدام النماذج المجانية مباشرة دون الحاجة إلى مفتاح API حقيقي. النظام يستخدم تلقائيًا النماذج المجانية عندما:

  1. لا يتم توفير مفتاح API
  2. يتم تحديد نموذج مجاني صراحةً (يحتوي على :free في اسمه)

النماذج المجانية المتاحة:

  • google/gemini-2.5-pro-exp-03-25:free - نموذج Google Gemini 2.5 Pro
  • qwen/qwen2.5-vl-72b-instruct:free - نموذج Qwen 2.5 (جيد للغة العربية)
  • meta-llama/llama-3.3-70b-instruct:free - نموذج Meta Llama 3.3

كيفية استخدام النماذج المدفوعة

لاستخدام النماذج المدفوعة، تحتاج إلى:

  1. الحصول على مفتاح API من OpenRouter
  2. إضافة المفتاح إلى ملف .env في المشروع
# في ملف .env
OPENROUTER_API_KEY=sk-or-v1-your-actual-key-here

النماذج المدفوعة المتاحة:

  • openai/gpt-4 - نموذج OpenAI GPT-4
  • anthropic/claude-3-opus - نموذج Anthropic Claude 3 Opus
  • anthropic/claude-3-sonnet - نموذج Anthropic Claude 3 Sonnet
  • meta-llama/llama-3-70b-instruct - نموذج Meta Llama 3 70B
  • mistralai/mistral-large - نموذج Mistral Large

كيفية تغيير النموذج المستخدم

يمكنك تغيير النموذج المستخدم بإحدى الطريقتين:

1. من خلال واجهة المستخدم

أرسل الأمر التالي في نموذج الاستعلام:

تغيير نموذج إلى google/gemini-2.5-pro-exp-03-25:free

2. من خلال ملف .env

قم بتعديل المتغير ADVANCED_AI_MODEL في ملف .env:

ADVANCED_AI_MODEL=google/gemini-2.5-pro-exp-03-25:free

للمطورين: استخدام OpenRouter في الكود

يمكنك استخدام OpenRouter مباشرة في الكود الخاص بك باستخدام مكتبة OpenAI:

from openai import OpenAI

client = OpenAI(
    base_url="https://openrouter.ai/api/v1",
    api_key="YOUR_OPENROUTER_API_KEY",
)

completion = client.chat.completions.create(
    model="google/gemini-2.5-pro-exp-03-25:free",  # نموذج مجاني
    messages=[
        {
            "role": "user",
            "content": "اشرح لي كيفية استخدام OpenRouter"
        }
    ],
    extra_headers={
        "HTTP-Referer": "https://your-site.com",  # اختياري
        "X-Title": "MyProjectName",               # اختياري
    }
)

print(completion.choices[0].message.content)

ملاحظات هامة

  • المفتاح المجاني يعمل فقط مع النماذج المجانية المحددة
  • تأكد من كتابة اسم النموذج بدقة كما هو موجود في OpenRouter
  • يتم تحديث قائمة النماذج المجانية من وقت لآخر، تحقق من وثائق OpenRouter للحصول على أحدث المعلومات
